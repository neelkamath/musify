# Musify

Musify is a web app which lets you edit your music playlists. This is my project for [PESU](http://pes.edu/)'s UE18CS355 course.

## Installation

1. Install the [app](docs/install.md).
1. `npm run build`

## Usage

Serve the website which has been saved to `dist/`.
